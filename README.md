# 📄 PDF Q&A Chatbot (RAG)

A chat-based PDF Question–Answering application built using:

- Streamlit
- LangChain
- FAISS
- Sentence Transformers
- Groq LLM

## 🚀 Features
- Upload and process PDF documents
- Chunking + embeddings (FAISS)
- Retrieval-Augmented Generation (RAG)
- Chat interface with memory
- Fast responses using Groq

## 🛠 Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- Groq API

## ▶ Run Locally

```bash
git clone https://github.com/omkargend455/pdf-rag-chatbot.git
cd pdf-rag-chatbot
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
